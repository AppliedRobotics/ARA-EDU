<h1 style="text-align: center;">Diff для обеспечения работы симулятора AgroTechSim с комплектом учебного оборудования для отработки навыков автономных полетов </h1>

### Что такое diff?
**diff** (от слова difference - разница) - это команда CLI, которая выводит все измененные параметры (разницу между предустановленными параметрами и настройками, внесенными пользователем)

### Где и как посмотреть diff
- Откройте Конфигуратор и зайдите во вкладку **Командная строка**
- Введите команду ```diff```
### Как сохранить diff?
- Откройте Конфигуратор и зайдите во вкладку **Командная строка**
- Введите команду ```diff```
- После выводы команды нажмите **Сохранить в файл**
- После сохранения убедитесь, что расширение файла .txt, если требуется, измените его
### Как загрузить готовый diff?
- Откройте Конфигуратор и зайдите во вкладку **Командная строка**
- Введите команду ```defaults``` (данная команда сотрет все имеющиеся настройки)
- Дождитесь перезагрузки
- Зайдите во вкладку **Командная строка**
- Нажмите **Загрузить из файла**
- Нажмите **Выполнить** и дождитесь перезагрузки

Для работы ARA EDU и Комплекта учебного оборудования для развития навыков автономных полетов используются 2 diff-файла, для каждого режима работы:

```
dif_educational-set-sim_mix1_XX-XX-XX.txt
dif_educational-set-sim_mix2_XX-XX-XX.txt

где XX-XX-XX - дата последнего обновления
```
 
### Как загрузить настройки для нескольких профилей микшера
Для осуществления полетов в синхронизации с симулятором на учебном оборудовании необходимо обновить настройки полетного контроллера.

Стабильный и четкий полет осуществляется с помощью переключения между разными режимами полета (ANGLE и ACRO) и для этого нужно загрузить diff для каждого профиля микшера.

- Откройте Конфигуратор и зайдите во вкладку **Командная строка**
- Введите команду ```defaults``` (данная команда сотрет все имеющиеся настройки)
- Дождитесь перезагрузки
- Убедитесь, что у вас открыт 1 профиль микшера (верхний правый угол, рядом с кнопкой "Отключить")
- Зайдите во вкладку **Командная строка**
- Нажмите **Загрузить из файла** и выберите diff для 1 профиля
- Нажмите **Выполнить** и дождитесь перезагрузки
- Переключитесь во 2 профиль микшера
- Зайдите во вкладку **Командная строка**
- Нажмите **Загрузить из файла** и выберите diff для 2 профиля
- Нажмите **Выполнить** и дождитесь перезагрузки
